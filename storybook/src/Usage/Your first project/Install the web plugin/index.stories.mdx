# Install the web plugin

To see the list of plugins provided by the framework run the command.

```jsx
> node glue plugin-list
```

The above command will list all the plugins provided by the framework.

```jsx
┌───────────┬────────────────────────┬────────────────────────────────────────┐
│  (index)  │        package         │              description               │
├───────────┼────────────────────────┼────────────────────────────────────────┤
│    web    │    '@gluestack/web'    │    'Installs a Next.js App Plugin'     │
│  mobile   │  '@gluestack/mobile'   │  'Installs a React Native App Plugin'  │
│ storybook │ '@gluestack/storybook' │   'Installs a Storybook App Plugin'    │
│ functions │ '@gluestack/functions' │  'Installs a Function for computing'   │
│  graphql  │  '@gluestack/graphql'  │ 'Installs a Hasura GraphQL App Plugin' │
│   auth    │   '@gluestack/auth'    │      'Installs auth in your App'       │
│  storage  │  '@gluestack/storage'  │     'Installs storage in your App'     │
│   cron    │   '@gluestack/cron'    │      'Installs Cron in your App'       │
│   jobs    │   '@gluestack/jobs'    │      'Installs Jobs in your App'       │
└───────────┴────────────────────────┴────────────────────────────────────────┘
```

The add any plugin from the above list run the command `node glue add <plugin-index-name>`.

So to install the web plugin to your project run the glue command.

```jsx
> node glue add web <web-project-name>
```

The above command will add a new folder `<web-project-name>` to your project folder.

The web plugin is the latest NextJS application with added features on top of it. For more details refer to `{link_to_web_plugin}`
